Creating postgres container:
1. docker run --name reactivities-postgres -e POSTGRES_USER=admin -e POSTGRES_PASSWORD=secret -p 5432:5432 -d postgres:latest

Deploy:
1. Delete wwwroot
2. In client-app/ run `npm run build`
3. Commit changes
4. docker lolgin
5. docker build -t <docker-username>/reactivities .

    Copy "ConnectionStrings" and "TokenKey" to appsettings.json
    At "DefaultConnection" -> Server: Replace localhost with host.docker.internal
    docker run --rm -it -p 8080:80 tudornicoara/reactivities
    Check localhost:8080
6. docker push  <docker-username>/reactivities:latest
